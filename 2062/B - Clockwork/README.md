In each second, the following happens in order:</p><ul> <li> Each clock's time decreases by $$$1$$$. If any clock's time reaches $$$0$$$, you lose immediately. </li><li> You can choose to move to an adjacent clock or stay at the clock you are currently on. </li><li> You can reset the time of the clock you are on back to its initial value $$$a_i$$$. </li></ul><p>Note that the above events happen in order. If the time of a clock reaches $$$0$$$ in a certain second, even if you can move to this clock and reset its time during that second, you will still lose.</p><p>You can start from any clock.
