In the following examples, the median letter is highlighted: <span class="tex-font-style-tt">con<span class="tex-font-style-bf"><span class="tex-font-style-underline">t</span></span>est</span>, <span class="tex-font-style-tt">i<span class="tex-font-style-bf"><span class="tex-font-style-underline">n</span></span>fo</span>.
